FAQ

How do I install the template?

To install the template, simply copy and paste the items in the code folder to your designated location on your server. 

 

How do I edit the mail address on the contact form?

There is only one PHP file in the pack. Locate it and edit it, and replace the email address there with your address. 

 

What are the i classes, those are icons? How do I change them?

The i classes are icons, you can change them by visiting http://fortawesome.github.io/Font-Awesome/icons/ font awesome and checking out the icons avaiable! If you want to update the font-awesome script in the future, all you need to do is replace the contents of the fonts folder and the font-awesome.css with the newest version that will be released on the FontAwesome website.

Navigation

Creating a simple menu item

    <div class="navigation">
        <ul>
            <li>
             <a class="active-menu" href="index.html">
                 <i class="fa fa-home"></i>
                    Home
                    <i class="fa fa-circle"></i>
                </a>
            </li>
        </ul>
    </div>

 

Inside the navigation div, inside the ul class, create a list like the example above. Adding active-menu to the inner anchor will make the menu bold, and the small circle to the right will become green

Creating a inactive dropdown menu 

<div class="navigaton">
     <ul>
         <li>
             <a class="show-sub-menu" href="#">
                 <i class="fa fa-cog"></i> 
                 Features 
                 <i class="fa fa-plus"></i>
             </a>
             <ul class="sub-menu">
                 <li><a href="features-jquery.html"><i class="fa fa-angle-right"></i>jQuery<i class="fa fa-circle"></i></a></li>
                 <li><a href="features-others.html"><i class="fa fa-angle-right"></i>Others<i class="fa fa-circle"></i></a></li>
             </ul>
         </li>
     </ul>
</div>

 

Inside the navigation div, inside the ul class, create a list like the one above. Line 4 5 6 7 8 are the visible menu. The one that gets tapped to deploy. 

Line 9 houses the submenu elements, the elements shown after tapping the main menu. These are a simple list, inside, there is an anchor housing an icon, a text, and the final icon. You can replace the icon and by using the font awesome refference chart. 

This menu will show up as closed. You will have to tap the menu to make it visible 

Creating an active submenu ( deployed ) 

 

<div class="navigaton">
     <ul>
         <li>
             <a class="show-sub-menu" href="#">
                 <i class="fa fa-cog"></i> 
                 Features 
                 <i class="fa fa-plus rotate-submenu-icon"></i>
             </a>
             <ul class="sub-menu show-submenu-item">
                 <li><a class="active-menu" href="features-jquery.html"><i class="fa fa-angle-right"></i>jQuery<i class="fa fa-circle"></i></a></li>
                 <li><a href="features-others.html"><i class="fa fa-angle-right"></i>Others<i class="fa fa-circle"></i></a></li>
             </ul>
         </li>
     </ul>
</div>

 

The only difference that applies to a menu that is active, is that, on line 7, the i class must have rotate-submenu-icon added to it, this will turn the plus sign, into a red x sign. You must also add on line 9 the class show-submenu-item, to make the list visible. That means when you open the main menu, the list will be visible instantly. 

To make any item from the submenu show up as active, add the class active-menu to any anchor within submenus or simple menus. 

 

Creating Boxed and Unboxed elements. 

Every element in the template is housed in a container, as explained in the second documentation. 

A container is a box, that adds a margin bottom to maintain visual beauty to the template. 

<div class="container">
     Your content here
</div>

This will create a simple container, with no box background

<div class="container boxed">
     your content here
</div>

This will create a simple container, with a boxed background.

What elements work with boxed containers?

All elements work with boxed containers, but please remember! This is very important! Small screens have a 320 pixel width available. The content class adds a 20 pixels to each side as a bleed margin, so you are left with 280 pixels, and boxes add another 20 pixels to each side as bleed margins so the total space left to add content is 240 pixels. This isn't bad if you use your content in a single container or in one-half/third-responsive boxes, but using more than 2 columns in a boxed mode, may make them look very tight on small mobiles devices such as the iPhones ( 2, 3G, 3GS 4, 4S, 5, 5S ).

It's recommended that boxes be used in the following system 

<div class="one-half-responsive boxed">
    Your Content Here
</div>

<div class="one-half-responsive boxed last-column">
    Your Content here
</div>

This means that boxes will show as rows on mobiles and as two columns on tablets ( you can use one-third as well, please check second documentation ) 

 

 

 

 

 

Sliders

    <div class="slider-container full-bottom">
        <div class="homepage-slider" data-snap-ignore="true">                
            <div>
                <div class="overlay"></div>
                <div class="homepage-slider-caption">
                    <h3>Swipe me!</h3>
                    <p>Hope you enjoy our latest item!</p>
                </div>
                <img src="images/pictures/5ww.jpg" class="responsive-image" alt="img">
            </div>
        </div>
    </div>

LIne 1 creates the image slider and adds the bottom margin

Line 2 creates the homepage slider wrapper the holds all the elements within it 

Line 3,4,5,6,7,8,9,10 is an image slide

Line 4 is the slider overlay. This is the black background over which the text sits

Line 5 is the homepage slider caption wrapper 

Line 6 is the main title

Line 7 is the small text

Line 8 closes the caption

Line 9 is the image. 

To create another slide, duplicate lines 3 to 10 and replace the text and image with your required ones

 

All sliders in the template follow a repetative pattern. A main div class that holds other divs with text or images in it. If you duplicate the class that holds the individual item and replace the values with yours, you will achieve  anew slide with great ease!
